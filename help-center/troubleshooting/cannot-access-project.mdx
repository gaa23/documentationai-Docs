---
title: Cannot Access Project
description: Fix project access issues in MyDose.
---

## Symptoms

- "Access Denied" or project not visible
- Invited users can't enter project

## Causes

- Wrong email address
- Invitation link expired
- Insufficient permissions

## Solution
<Steps>
  <Step title="Check Email">
    Make sure you're using the invited email address to sign in.
  </Step>
  <Step title="Resend Invite">
    Ask your admin to resend the invitation from the Team tab.
  </Step>
</Steps>
