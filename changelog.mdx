<Update label="2024-10-15" description="v2.1.0" tags={["feature", "improvement"]}>

## New Features

- Added AI-powered biomarker trend analysis for clinics, providing predictive insights on patient progress.
- Introduced patient self-scheduling for telehealth appointments directly from the dashboard.

## Improvements

- Enhanced protocol builder with drag-and-drop interface for faster treatment plan creation.
- Optimized lab result uploads, reducing processing time by `>50%`.

## Bug Fixes

- Fixed intermittent sync issues between clinic EMR systems and MyDose patient records.
- Resolved mobile view rendering problems in the patient portal.

</Update>

<Update label="2024-09-25" description="v2.0.0" tags={["feature", "breaking"]}>

## New Features

- Launched unified patient portal with real-time lab tracking and personalized longevity recommendations.
- Integrated telehealth video calls with automatic note generation via AI transcription.

## Breaking Changes

- Updated API authentication to require `{API_KEY}` in headers only (removed query param support).

<CodeGroup tabs="JavaScript,cURL">
```javascript
// New header-only auth
const response = await fetch('https://api.mydose.ai/v2/patients', {
  headers: {
    'Authorization': `Bearer ${YOUR_API_KEY}`
  }
});
```

```bash
# New header-only auth
curl -H "Authorization: Bearer YOUR_API_KEY" https://api.mydose.ai/v2/patients
```
</CodeGroup>

## Bug Fixes

- Corrected protocol adherence reporting discrepancies in multi-clinic setups.

</Update>
